1. Quem somos

Para dúvidas ou solicitações relacionadas aos seus dados pessoais, entre em contato pelo e-mail: drajuliahoss@gmail.com

2. Quais dados coletamos

Coletamos apenas os dados necessários para viabilizar o contato e o agendamento de consultas. Isso inclui:

  • Nome, quando fornecido voluntariamente via formulário ou mensagem
  • Número de telefone ou WhatsApp, quando você inicia contato por esse canal
  • Endereço de e-mail, quando utilizado para contato
  • Dados de navegação anônimos (páginas acessadas, tempo de visita), coletados automaticamente para fins de desempenho do site

Não coletamos dados de saúde, documentos de identificação ou informações financeiras por meio deste site.

3. Para que usamos seus dados

Os dados fornecidos são utilizados exclusivamente para:

  • Responder às suas mensagens e dúvidas
  • Agendar e confirmar consultas
  • Enviar informações que você solicitou sobre tratamentos

Não utilizamos seus dados para envio de publicidade não solicitada, nem os compartilhamos com terceiros para fins comerciais.

4. Base legal para o tratamento

O tratamento dos seus dados é realizado com base nas seguintes hipóteses previstas na Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018):

  • Consentimento: quando você preenche um formulário ou inicia contato voluntariamente
  • Legítimo interesse: para responder às suas solicitações e viabilizar o atendimento
  • Execução de contrato: quando há agendamento confirmado de consulta

5. Com quem compartilhamos seus dados

Seus dados podem ser processados pelas seguintes plataformas, utilizadas para operação do site e comunicação:

  • WhatsApp Business — canal de atendimento e agendamento (Meta Platforms)
  • Google — ferramentas de análise de desempenho de campanhas (Google Analytics e Google Ads)
  • GitHub Pages — hospedagem das páginas de agendamento

Nenhum dado é vendido ou cedido a terceiros para fins publicitários.

6. Por quanto tempo guardamos seus dados

Os dados de contato são mantidos pelo tempo necessário para o atendimento solicitado. Dados de pacientes em tratamento são mantidos conforme exigido pela legislação sanitária e pelo Código de Ética Odontológica.

Dados de navegação anônimos são retidos conforme a política do Google Analytics.

7. Seus direitos como titular dos dados

Conforme a LGPD, você tem direito a:

  • Confirmar se tratamos dados seus
  • Acessar os dados que temos sobre você
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a exclusão dos seus dados, quando aplicável
  • Revogar seu consentimento a qualquer momento
  • Solicitar informações sobre com quem compartilhamos seus dados
Para exercer qualquer um desses direitos, entre em contato pelo e-mail drajuliahoss@gmail.com. Respondemos em até 15 dias úteis.

8. Cookies e rastreamento

O site utiliza cookies técnicos necessários para seu funcionamento. São utilizados cookies de análise de desempenho do Google Analytics e Google Ads, que coletam dados de forma anonimizada para mensurar o desempenho das campanhas.

Você pode configurar seu navegador para recusar cookies, mas isso pode afetar o funcionamento de algumas partes do site.

9. Segurança dos dados

Adotamos medidas técnicas e organizacionais adequadas para proteger seus dados contra acesso não autorizado, perda ou uso indevido. O site opera exclusivamente sob protocolo HTTPS.

10. Alterações nesta política

Esta política pode ser atualizada periodicamente. A data da última revisão está indicada no topo da página. Recomendamos que você consulte esta página regularmente.
